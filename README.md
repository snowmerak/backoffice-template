# Backoffice Template

This is a simple template for back office applications.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/) (>= 8.9.0)
- [Yarn](https://yarnpkg.com/en/)

### Installing

```bash
git clone https://github.com/snowmerak/backoffice-template.git
cd backoffice-template
yarn
```
